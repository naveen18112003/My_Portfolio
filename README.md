## üöÄ How to Run Locally

Follow these steps to set up the portfolio on your local machine:

### Prerequisites
- [Node.js](https://nodejs.org/) (v18.0.0 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### 1. Clone the Repository
```bash
git clone https://github.com/naveen18112003/Autonomous-AI-Business-Operations-Manager.git
cd "My Portfolio"
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to see the live site with Hot Module Replacement (HMR).

### 4. Build for Production
To generate an optimized production build:
```bash
npm run build
npm run start
```
